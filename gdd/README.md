# Game Design Document - Overview

This directory contains the Game Design Documents for the Mining Game V2.

## Structure

- [01_Introduction](01_introduction.md): High-level concept and vision.
- [02_Gameplay_Mechanics](02_gameplay_mechanics.md): Detailed mechanics, controls, and loops.
- [03_Technical_Specification](03_technical_specification.md): Technical architecture and Godot-specific implementation details.
- [04_Visuals_and_Audio](04_visuals_and_audio.md): Art style, UI/UX, and sound design.

## Maintenance

This GDD is a living document. Any requirement changes requested by the user will be reflected here first before implementation begins.
